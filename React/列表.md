# React列表
## 在组件中渲染列表数据是非常常见的场景
``` 
class PostList extends Component{
    render(){
        return(
            <div className = 'container'>
                  <h2>帖子列表</h2>
                  <ul>
                        {this.state.posts.map(
                            item => 
                            <PostItem 
                            post = {item}
                            onVote = {this.handleVote}
                            />
                        )}
                  </ui>
            </div>
        );
    }
}
```
*上述代码在浏览器控制台中会报错,原因是列表中的每个元素没有key的属性*

## Key的作用

*React使用key属性来标记列表中的每个元素,当列表数据发生变化时,React就可以通过key知道哪些元素发生了变化,从而只重新渲染发生变化的元素,提高渲染效率.*

``` 
class PostList extends Component{
    render(){
        return(
            <div className = 'container'>
                  <h2>帖子列表</h2>
                  <ul>
                  {/*将id赋值给key属性,作为唯一标识符*/}
                        {this.state.posts.map(
                            item => 
                            <PostItem 
                            key = {item.id}
                            post = {item}
                            onVote = {this.handleVote}
                            />
                        )}
                  </ui>
            </div>
        );
    }
}
```

*当没有id,直接使用默认索引*
``` 
class PostList extends Component{
    render(){
        return(
            <div className = 'container'>
                  <h2>帖子列表</h2>
                  <ul>
                        {this.state.posts.map(
                            (item,index) => 
                            <PostItem 
                            key = {index}
                            post = {item}
                            onVote = {this.handleVote}
                            />
                        )}
                  </ui>
            </div>
        );
    }
}
```
